# webdriverio
Simple project using webdriverIO v7.3 with Mocha and Allure report

## Running the tests: 
```javascript
npm test -- --suite contactus
```

## Generating and opening Allure reports: 
```javascript
allure generate C:/devenv/webdriverioFramework/allure-results --clean
allure open
```
